# Node_GraphQL_Backend_Template
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
# watch mode
$ npm run start:dev
# production mode
$ npm run start:prod
```

- Open browser and go to http://localhost:3001/graphql, you can check your API here.

## Test

```bash
# unit tests
$ npm run test
# e2e tests
$ npm run test:e2e
# test coverage
$ npm run test:cov
```

## Deploy your service

```bash
# Run command:
$ yarn start-docker


# Stop your service:
$ yarn stop-docker
```
